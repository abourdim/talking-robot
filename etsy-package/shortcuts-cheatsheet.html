Talking Robot — Buttons & Voice Controls

Cheat Sheet — Laminate Me!
Print → Laminate → Stick near the keyboard
Main Buttons
SpeakReads the text out loud (mouth animates)
ListenMic on → speech becomes transcript
StopCancels the current speech
ConnectPair a BBC micro:bit V2 over Bluetooth
DisconnectRelease the micro:bit bridge
Voice & Language Picker
Voice ▾Pick any system voice (deep, bright, etc.)
Language ▾Match the voice to its language tag
Rate / PitchSlow the robot down, pitch it up or down
Auto-replyToggle: after Listen, robot speaks back

Keyboard Shortcut
Ctrl+EnterIn the text field → Speak
Cmd+EntermacOS equivalent of the above
That's the only built-in shortcut — everything else is a button you click.
Optional micro:bit Bridge
SendPush current text to the paired micro:bit
Once connected, typed words can auto-stream to the board — great for wearables, badges, or room displays.
Pro Tips
Browser support: Chrome or Edge only (desktop or Android). Microphone permission needed for Listen; Bluetooth only if you use the optional micro:bit bridge.
