الوصف الكامل

🤖 Talking Robot — تكلّم • استمع • عبّر. روبوت داخل المتصفّح للأطفال والصفوف والصنّاع الفضوليّين. افتح ملف HTML واحدًا في Chrome أو Edge ليظهر لك روبوت لطيف ينطق كلماتك بصوت عالٍ، ويستمع عبر الميكروفون، ويتفاعل بعشرة وجوه مشاعر بنقرة واحدة. اختياريًّا: جسر إلى BBC micro:bit V2 عبر Web Bluetooth ليستقبل كلماتك المكتوبة. ما ستحصل عليه • 🗣️ تكلّم — اكتب نصًّا، اختر صوتًا ولغة، اضغط «تكلّم». يتحرّك الفم مع واجهة Speech Synthesis. • 🎤 استمع — انقر «استمع»، تكلّم في الميكروفون، يظهر النصّ عبر واجهة Speech Recognition. خيار ردّ تلقائيّ. • 😄 10 وجوه مشاعر — 😐 محايد · 😄 سعيد · 🥺 حزين · 😠 غاضب · 😂 ضاحك · 😜 مرح · 😱 مصدوم · 🥳 احتفال · 🤖 روبوت · 😴 نعسان. • 📡 جسر بلوتوث اختياريّ — أقرن BBC micro:bit V2 لتصل كلّ كلمة مكتوبة إلى اللوحة عبر Web Bluetooth. • ⌨️ اختصار لوحة مفاتيح واحد — Ctrl/Cmd+Enter داخل مربّع النصّ يشغّل «تكلّم». • 🌐 يعمل محلّيًّا — بلا خادم، بلا تثبيت، بلا حساب. افتح index.html فقط. محتوى الملف المضغوط • index.html — التطبيق • app.js, styles.css, logo.svg — واجهة الروبوت ومنطقه • ble-uart.js — جسر البلوتوث الاختياريّ • makecode.ts — مخطّط الـ micro:bit المرفق • README.md + USERGUIDE.html — أدلّة كاملة • etsy-package/ — بطاقة بدء سريع قابلة للطباعة، ورقة اختصارات، خطّة درس، ملصق صفّ، ورقة ملصقات المتطلّبات • Chrome أو Edge (Web Speech API مطلوب — Safari و iOS غير مدعومَين) • ميكروفون لميزة «استمع» · سمّاعات لميزة «تكلّم» • اختياريّ: BBC micro:bit V2 (لجسر البلوتوث فقط) لمن هذا المنتج؟ معلّمون يُقدّمون دروس الصوت والذكاء الاصطناعيّ وعلوم STEM · عائلات التعليم المنزليّ · تدريب النطق واللغة · أطفال يريدون صديقًا آليًّا · صنّاع يحتاجون واجهة صوتيّة لمشروع micro:bit. خصوصيّة أوّلًا كلّ شيء يعمل داخل متصفّحك. بلا سحابة، بلا حسابات، بلا تتبّع. تحديثات مدى الحياة كلّ إصدار v1.x من هذه القائمة لك للأبد. تكلّم • استمع • عبّر 🤖
