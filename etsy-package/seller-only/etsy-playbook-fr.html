Description complète

🤖 Talking Robot — un robot vocal dans le navigateur qui parle, écoute et ressent. Ouvrez un seul fichier HTML dans Chrome ou Edge. Tapez une phrase, appuyez sur Parler — le visage robot la prononce avec une bouche animée par l'amplitude réelle de la parole. Appuyez sur Écouter et il transcrit ce que vous dites, puis répond. 13 émotions, 16 locales vocales, et un BBC micro:bit V2 optionnel qui reflète l'émotion sur ses LED 5×5. CE QUE VOUS OBTENEZ • 🗣 Taper + Parler — TTS via Web Speech avec débit, hauteur et voix ajustables • 🎙 Mode Écoute — reconnaissance Web Speech qui capte votre voix et la route dans le pipeline du robot • 😄 13 émotions animées sur un visage CSS expressif (neutre, content, triste, en colère, furieux, surpris, rire, silly, effrayé, fête, robot, nerdy, endormi) • 👀 Yeux qui suivent la souris + bouche animée par l'amplitude du signal TTS • 🌍 16 locales vocales : en-US, en-GB, fr-FR, es-ES, de-DE, it-IT, ja-JP, ko-KR, zh-CN, plus 8 arabes régionales (ar-SA, ar-EG, ar-MA, ar-TN, ar-DZ, ar-IQ, ar-JO, ar-LB) • 🔧 Bascules : parole auto, envoi auto au micro:bit, écoute continue, voix aléatoire, curseur sensibilité bouche • 📡 Pont BBC micro:bit V2 optionnel en BLE UART — jetons N_word par syllabe + commandes EMO:<nom> affichent l'émotion sur la matrice LED (cœur pour content, ZZZ pour endormi, visage fâché, etc.) • 📱 PWA — installable, fonctionne hors ligne en mode tapé CONTENU DU TÉLÉCHARGEMENT • index.html — l'application complète • makecode.ts — firmware micro:bit optionnel (à flasher une fois depuis MakeCode) • SETUP.md — démarrage rapide en 5 minutes • docs/ — guide illustré, aide-mémoire, FAQ, playbook Etsy • LICENSE — licence mono-utilisateur personnelle / classe PRÉREQUIS • Chrome ou Edge sur ordinateur ou Android (Web Speech API requis) • Microphone pour le mode Écoute • Optionnel : BBC micro:bit V2 + Bluetooth pour le miroir LED (voix + visage fonctionnent sans) POUR QUI Orthophonistes · enseignants de langues · classes de contes · parents d'enfants bilingues · écoles arabophones cherchant des voix régionales · clubs STEM voulant un visage amical pour micro:bit. CONFIDENTIALITÉ D'ABORD Rien ne quitte votre ordinateur. Pas d'analytique, pas de connexion, pas de cloud. La reconnaissance vocale passe par l'API Web Speech du navigateur ; les réglages restent dans localStorage. MISES À JOUR À VIE Chaque version publiée depuis cette annonce est à vous pour toujours. Retéléchargez le dernier ZIP à tout moment depuis votre page Achats Etsy. Parler · Écouter · Ressentir · Connecter. 🤖🗣🎙
