5 Listing FAQ (add inside the description or in a separate section)

Q: Do I need to know how to code?

No — unzip, open index.html in Chrome/Edge, type or talk. That's it.

Q: Will this work on my iPad?

No. Apple does not allow the Web Speech API or Web Bluetooth in iOS browsers. Use a laptop, Chromebook, or Android tablet.

Q: Can I use this in my classroom of 30 students?

Yes — single teacher, one school, unlimited students. For multi-teacher / district licenses, message me.

Q: Do you offer refunds?

Etsy doesn't allow refunds on instant digital downloads, so please read the requirements carefully before purchasing.

Q: Will it get updates?

Free bug-fix updates for 12 months — message me with your Etsy order number.
