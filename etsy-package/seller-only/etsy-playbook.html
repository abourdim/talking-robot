Full description

🤖 Talking Robot — Talk • Listen • Emote. A browser robot for kids, classrooms, and curious makers. Open one HTML file in Chrome or Edge and you have a friendly robot that speaks your words out loud, listens through your microphone, and reacts with 10 one-click emotion faces. Optional: bridge to a BBC micro:bit V2 over Web Bluetooth to stream your typed words to the board. WHAT YOU GET • 🗣️ TALK — type text, pick a voice / language, click Speak. The mouth animates in sync with the browser Speech Synthesis API. • 🎤 LISTEN — click Listen, speak into the mic, the transcript appears via the browser Speech Recognition API. Optional auto-reply so the robot speaks back. • 😄 10 EMOTION FACES — 😐 neutral · 😄 happy · 🥺 sad · 😠 angry · 😂 laughing · 😜 silly · 😱 shocked · 🥳 party · 🤖 robot · 😴 sleepy. One click changes the mood. • 📡 OPTIONAL BLUETOOTH BRIDGE — pair a BBC micro:bit V2 and every word you type streams to the board over Web Bluetooth. • ⌨️ ONE KEYBOARD SHORTCUT — Ctrl/Cmd+Enter in the text field triggers Speak. • 🌐 RUNS LOCALLY — no server, no install, no account. Just open index.html. WHAT'S IN YOUR DOWNLOAD • index.html — the app • app.js, styles.css, logo.svg — the robot UI & logic • ble-uart.js — the optional Bluetooth bridge • makecode.ts — the matching micro:bit sketch (if you use the bridge) • README.md + USERGUIDE.html — full walkthroughs • etsy-package/ — printable quick-start, buttons cheat sheet, lesson plan, classroom poster, sticker sheet REQUIREMENTS • Chrome or Edge (Web Speech API required — Safari & iOS are not supported) • A microphone for the Listen feature · speakers for TALK • Optional: BBC micro:bit V2 (for the Bluetooth bridge only) WHO IT'S FOR Teachers running voice / AI / STEM lessons · home-schooling families · speech-and-language practice · kids who want a robot friend · makers who need a speech front-end for a micro:bit project. PRIVACY FIRST Everything runs in your browser. No cloud, no accounts, no analytics. The transcript and the text you type never leave your device. LIFETIME UPDATES Every v1.x release shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page. Talk • Listen • Emote 🤖

Common mistakes

❌ Don't✅ Do
Start with "Hi, I'm…" + 10s of introOpen with the robot actually speaking in the first 3 s
Film landscape (16:9)9:16 portrait — Etsy autoplays on mobile
Talk over the robot's voiceLet the robot speak, then narrate in the quiet beat
Forget captions (Etsy plays muted)Burn in captions — especially on the Listen transcript scene
Show iOS / Safari in the videoAlways demo on Chrome / Edge — iPhone users will buy and return
Skip the CTASay "link below" in the last 5 seconds every single time
