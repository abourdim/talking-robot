Full description

🤖 Talking Robot — a browser-based voice robot that talks, listens, and feels. Open one HTML file in Chrome or Edge. Type a sentence, press Speak — the robot face speaks it with an animated mouth driven by real speech amplitude. Press Listen and it transcribes what you say, then answers back. 13 emotions, 16 voice locales, and an optional BBC micro:bit V2 that mirrors the emotion on its 5×5 LEDs. WHAT YOU GET • 🗣 Type + Speak — Web Speech TTS with adjustable rate, pitch, and voice • 🎙 Listen mode — Web Speech recognition captures your voice and routes it through the robot's pipeline • 😄 13 animated emotions on an expressive CSS robot face (neutral, happy, sad, angry, angry+, surprised, laugh, silly, scared, party, robot, nerdy, sleepy) • 👀 Eyes-follow-mouse + mouth animation amplitude-driven by the TTS waveform • 🌍 16 voice locales: en-US, en-GB, fr-FR, es-ES, de-DE, it-IT, ja-JP, ko-KR, zh-CN, plus 8 Arabic regionals (ar-SA, ar-EG, ar-MA, ar-TN, ar-DZ, ar-IQ, ar-JO, ar-LB) • 🔧 Toggles: auto-speak, auto-send to micro:bit, continuous listening, random voice, mouth sensitivity slider • 📡 Optional BBC micro:bit V2 bridge over BLE UART — per-syllable N_word tokens + EMO:<name> commands make the LED matrix show the matching emotion (heart for happy, ZZZ for sleepy, angry face, etc.) • 📱 PWA — installable, runs offline in typed mode WHAT'S IN YOUR DOWNLOAD • index.html — the full app • makecode.ts — optional micro:bit firmware (flash once from MakeCode) • SETUP.md — 5-minute quick-start • docs/ — illustrated guide, cheat sheet, FAQ, Etsy playbook • LICENSE — personal / classroom single-user license REQUIREMENTS • Chrome or Edge on desktop or Android (Web Speech API required) • Microphone for Listen mode • Optional: BBC micro:bit V2 + Bluetooth for the LED mirror (voice + face work without it) WHO IT'S FOR Speech therapists · language teachers · storytelling classrooms · parents of multilingual kids · Arabic-speaking schools looking for regional voices · STEM clubs wanting a friendly micro:bit face. PRIVACY FIRST Nothing leaves your computer. No analytics, no logins, no cloud. Speech recognition runs through the browser's Web Speech API; settings live in localStorage. LIFETIME UPDATES Every version shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page. Talk · Listen · Feel · Connect. 🤖🗣🎙
