Sample Lesson: Teach a Robot to React

Voice + emotion + Bluetooth — 45 minutes
Teach a robot to react: voice + emotion + Bluetooth
45 minutes
Ages 8–14
STEM / Computing / Language Arts
Learning Objectives
1. Use the browser Speech Synthesis API to make a robot speak a sentence.
2. Use Speech Recognition to capture a spoken phrase as text.
3. Match an emotion face to the meaning of a message.
4. (Stretch) Pair a BBC micro:bit V2 over Web Bluetooth and stream text to it.

Robot features used in this lesson

  • TALK type text, pick a voice, click Speak — mouth animates in sync
  • LISTEN click Listen, speak into the mic, transcript appears; auto-reply optional
  • 10 Emotion Faces 😐 😄 🥺 😠 😂 😜 😱 🥳 🤖 😴 — click to change instantly
  • Bluetooth Bridge (optional) BBC micro:bit V2 receives typed words live
  • Runs locally open index.html in Chrome/Edge, no install
