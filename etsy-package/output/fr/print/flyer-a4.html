TALKING-ROBOT
micro:bit V2
Fais parler les robots. 13 émotions. 16 voix.
Le navigateur EST le visage du robot.
Tape, parle, change de visage. Miroir émotion 5×5 LEDs sur un micro:bit (option).

Démarrez en 60 secondes

  1. Ouvre Chrome ou Edge.
  2. Tape ou parle une phrase.
  3. Choisis une émotion dans la grille.
  4. Le visage du robot s'anime et parle.
  5. (Optionnel) connecte un micro:bit pour le miroir LED.
🎭

13 émotions animées

Joyeux, farceur, surpris, fête, endormi, robot, nerd, amoureux, cool, fâché, triste, timide, clin d'œil.

🌍

16 locales vocales

Anglais, français, arabe — dont 8 variantes arabes (KSA, Égypte, Maroc, Tunisie, Algérie, Levant, Golfe, MSA). Rare.

🤖

Miroir BLE (option)

Connecte un micro:bit V2 : commandes EMO:name → miroir émotion 5×5 LEDs. Tokens N_word par syllabe.
